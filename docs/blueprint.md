# **App Name**: Cosecha Inteligente

## Core Features:

- User Authentication: Secure login and registration for producers and urban gardeners.
- Crop Catalog: Browse a catalog of crops with detailed information and recommendations specific to El Salvador.
- Cultivation Planning: Create personalized cultivation plans based on crop type, location, and planting date.
- Plan Visualization: View cultivation plans with timelines, key milestones, and actionable recommendations.
- Personalized recommendations: Get best practices advice about soil preparation or the time of seeding, to help ensure a higher success rate with generative AI based on crop and planting details, acting as a tool to reason when and how to offer helpful advice.
- User Profile Management: Manage user information, change password, and enable two-factor authentication.
- Admin Panel: Manage crops and user data, including editing existing crops and adding new ones.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23) to evoke agriculture and nature.
- Background color: Light beige (#F5F5DC), a very desaturated tint of green, for a natural and unobtrusive backdrop.
- Accent color: Muted brown (#8B4513), analogous to green, for clear visual contrast in CTAs and other callouts.
- Body and headline font: 'PT Sans' for a balance of modernity and warmth.
- Simple, clear icons representing fields, plants, calendars, and weather, emphasizing usability for rural users.
- Clean, modern, and clear layout with a practical focus, ensuring easy navigation and readability on both desktop and mobile devices. Prioritize responsive design for accessibility across different devices.
- Subtle animations to highlight important information and guide the user through key interactions.
